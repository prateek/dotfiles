Zsh Harness Comparison

This report compares the shell harnesses currently living in ~/dotfiles. It focuses on what each harness actually tests, where each one runs, and how much any two overlap.

The pairwise overlap numbers are based on shared coverage groups, not file size or code similarity. The current report compares active harness surfaces, including the main harness modes that replaced the old wrapper scripts.

At A Glance

Coverage Matrix

Rows are coverage groups. Columns are harnesses. This makes the shape of the overlap obvious before you dig into pairwise diffs.

Pairwise Overlap Matrix

The score is shared coverage groups divided by the union of both harnesses’ groups. Higher means they are testing more of the same ground.
